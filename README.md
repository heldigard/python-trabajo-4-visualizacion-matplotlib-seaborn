# Trabajo 4: Visualización de datos con Matplotlib y Seaborn

## Objetivo
Crear visualizaciones con Matplotlib y Seaborn para analizar el dataset de ventas de Superstore.

## Visualizaciones implementadas

### Matplotlib - Univariantes
- Histograma de distribución de ventas
- Diagrama de barras de ventas por categoría

### Seaborn - Univariantes
- Boxplot de beneficios por categoría
- Violinplot de descuentos por segmento

### Matplotlib - Bivariantes
- Scatter plot: Ventas vs Beneficios
- Gráfico de líneas: Evolución temporal de ventas

### Seaborn - Bivariantes
- Barplot agrupado: Ventas por Categoría y Segmento
- Regplot: Cantidad vs Ventas

### Seaborn - Multivariante
- Heatmap de correlación

### Subplots
- Figura con 4 visualizaciones:
  - Top 10 ciudades por ventas
  - Distribución por segmento
  - Beneficios por región
  - Ventas por categoría y sub-categoría
